# sadda.recipe

Reproducibility primitive: a context manager that links operations
inside it to a named `recipe_run` row, plus a generated `.py` script
at block exit. PROVISIONAL tier.

See the [quickstart](../quickstart.md#record-an-analysis-recipe) for
a worked example. Design rationale lives in the
[2026-05-22 DEVLOG entry "Recipes (F1)"](https://github.com/sadda-speech/sadda/blob/main/DEVLOG.md).
