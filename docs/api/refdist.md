# sadda.refdist

Reference distributions — install + query curated populations
(observed corpora, normative ranges, target zones); pin them per
project; publish your own back to the registry. PROVISIONAL tier.

Design rationale lives in the
[2026-05-18 "Reference distribution governance" DEVLOG entry](https://github.com/sadda-speech/sadda/blob/main/DEVLOG.md).
The three-tier registry, the `refdist.toml` schema, and the
GUI overlay encoding for observed vs normative vs target are all
covered there.
